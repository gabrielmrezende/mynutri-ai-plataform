# Roadmap do Projeto - MyNutri AI

Este documento descreve as fases de desenvolvimento do projeto.

---

## MVP (Produto Mínimo Viável)

Funcionalidades essenciais:

- Cadastro de usuário
- Login
- Questionário de anamnese
- Integração com IA
- Geração automática de dieta
- Exibição da dieta para o usuário

---

## Versão 2

Melhorias planejadas:

- Histórico de dietas
- Dashboard nutricional
- Ajuste automático de dieta
- Recomendações alimentares
- Melhor interface de usuário

---

## Versão 3

Expansões futuras:

- Aplicativo mobile
- Integração com smartwatch
- Monitoramento de progresso
- Recomendações de receitas
- Sistema de assinatura premium

---

## Futuro do Produto

Possíveis evoluções:

- Nutricionista digital baseado em IA
- Planos alimentares adaptativos
- Integração com APIs de alimentos
- Personalização avançada com aprendizado de máquina
